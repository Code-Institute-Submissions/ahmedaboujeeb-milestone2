About Memory Game

Memory Game is a very cool and simple game that tests your memory, Memory Game is also known as Match Up. Memory Game is a card game that consists of 16 cards and 8 colors, each 2 cards have a unique color.

Instructions

  • To start playing click the "Start" button.
  • Click on a card to see the card color.
  • Click on another card to find the matching card.
  • You can only open 2 cards at the same time.
  • You have 1 minute to match all cards.
  • When the minute is up, click to "Reset" to start over.
